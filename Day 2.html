Differences between var, let and const


const

it is introduced in ES6

it is block scoped

Example

const username = 'LAWAL ABDULGANIYU' // Declare and assign
username = Ade // cannot be reassigned
  • const cannot be re assigned and updated
  • const cannot be re-declared

var

    Var is the oldest way to declare variable

    it was used before ES6 and it is still supporteds

  • var can be re assigned and updated
  • var can be re-declared

Example

var x = 20;// Declare and assign
y = 2; //Declare and assign
x=17; //mutated
var y = 1; //re-declared

let

"let" was introduced in ES 6

  • it is block-scope
  • var can be mutated and updated
  • let cannot be re-declared

Example

let age = 25; // Declared and assigned
age = 25; // reassign or mutated
let age = 25; // cannot be re-declared
