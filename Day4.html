Differences between Comparison Operators and Logical Operators

Comparison Operators

These are used to compare two values, and the result of the comparison is either True or False.
These operators help determine relationships between values
Few types of Comparison are equal to(==),(===),(!=), (>=), greater than(>), etc...

Examples of comparison Operators

Example: 5 == 5 // evaluates to True because both values are equal.
Example: 4 == 5 // evaluates to False because the values are not equal.

Example: 3 < 6 // evaluates to True because 3 is less than 6.
Example: 10 < 4 // evaluates to False because 10 is not less than 4.

Logical Operators

Logical operators are used to combine two or more conditions and return a Boolean value (true or false) based on the operands.

Types and examples of Logical Operators
  • Logical AND (&&): Returns true if *both* operands are true; otherwise, returns false.
    Example:
    let a = true && true; // a is true
    let b = true && false; // b is false
  • Logical OR (||):
    Returns true if *at least one* of the operands is true; returns false only if both are false.
    Example:
    let a = true || false; // a is true
    let b = false || false; // b is false
  • Logical NOT (!):
    Reverses the Boolean value of the operand. If the operand is true, it returns false, and if it's false, it returns true.
    Example:
    let a = !true; // a is false
    let b = !false; // b is true
